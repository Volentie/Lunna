# Lunna

## Getting Started
To build the place from scratch, use:

```bash
rojo build -o "Lunna.rbxlx"
```

Next, open `Lunna.rbxlx` in Roblox Studio and start the Rojo server:

```bash
rojo serve
```

For more help, check out [the Rojo documentation](https://rojo.space/docs).
